Getting started with ivppmlhdfe

ivppmlhdfe fits instrumental-variable Poisson pseudo-maximum-likelihood (IV-PPML) models with high-dimensional fixed effects. It is an R translation of the Stata command of the same name (Kwon, Larch, Yoon and Yotov, 2026), and reproduces ppmlhdfe / fixest::fepois point estimates to machine precision in the plain (no-instrument) case.

Attribution. This package is only a translation. All credit for the estimator and methodology goes to the original authors, Ohyun Kwon, Mario Larch, Jangsu Yoon and Yoto V. Yotov. The R translation is by Guewen Heslan (PhD Candidate in Economics, Nantes Université, France) using Claude Opus, available at https://github.com/gheslan/ivppmlhdfe-for-r.

library(ivppmlhdfe)
data(ivppmlhdfe_ClassA)

The endogeneity problem

In the Monte-Carlo design the endogenous regressor x1 has a true coefficient of 0.5, but it is correlated with the error, so ordinary PPML is biased:

ppml <- ivppmlhdfe(y ~ x1 + x2 | id + year, data = ivppmlhdfe_ClassA)
coef(ppml)["x1"]   # biased upward, away from 0.5
#>        x1 
#> 0.7924923

Instrumenting x1 with z removes the bias:

iv <- ivppmlhdfe(y ~ x2 | id + year | x1 ~ z, data = ivppmlhdfe_ClassA)
coef(iv)["x1"]     # close to the true 0.5
#>        x1 
#> 0.4994833
summary(iv)

Two- and three-way gravity models

data(ivppmlhdfe_ClassB)
b <- ivppmlhdfe(y ~ x2 | exp^year + imp^year | x1 ~ z,
                data = ivppmlhdfe_ClassB, cluster = ~pair)
coef(b)["x1"]
#>        x1 
#> 0.4660402

^ denotes an interacted fixed effect (Stata’s #). Standard errors can be heteroskedasticity-robust (the default) or one-/two-way cluster-robust via the cluster argument.

Split-panel jackknife bias correction

Replacing the regressor with an instrument re-introduces incidental-parameter bias. The split-panel jackknife removes its leading-order term. For Class A the estimator splits along both the time and the cross-sectional dimension:

s <- ivppml_spj(y ~ x2 | id + year | x1 ~ z, data = ivppmlhdfe_ClassA,
                class = "A", id = "id", time = "year", B = 0)
s$b_full   # uncorrected
#>        x1 
#> 0.4994833
s$b_spj    # SPJ-corrected
#>        x1 
#> 0.5946432

Set B > 0 (the paper uses B = 1000) to obtain pair/individual-bootstrap standard errors and a percentile confidence interval.

Validating against ppmlhdfe / fepois

if (requireNamespace("fixest", quietly = TRUE)) {
  fe <- fixest::fepois(y ~ x1 + x2 | id + year, data = ivppmlhdfe_ClassA,
                       notes = FALSE)
  rbind(ivppmlhdfe = coef(ppml)[c("x1", "x2")],
        fepois     = coef(fe)[c("x1", "x2")])
}
#>                   x1        x2
#> ivppmlhdfe 0.7924923 0.3568299
#> fepois     0.7924923 0.3568299

The two sets of coefficients agree to machine precision.
